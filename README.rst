
===================
Leonardo Summernote
===================

A lightweight wysiwyg editor for Leonardo CMS

.. contents::
    :local:

Installation
------------

.. code-block:: bash

    pip install leonardo-summernote

or as leonardo bundle

.. code-block:: bash

    pip install django-leonardo["summernote"]

Sync static

.. code-block:: bash

    python manage.py sync_all

    # or
    
    python manage.py collectstatic --noinput
